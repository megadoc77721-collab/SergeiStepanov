# Photo.Design Portfolio

Статический premium-сайт-портфолио для дизайнера фотоконтента.

## GitHub Pages

Этот проект готов к публикации на GitHub Pages без сборки.

### Структура

- `index.html`
- `styles.css`
- `script.js`
- `.nojekyll`

### Публикация

1. Создайте новый публичный репозиторий на GitHub.
2. Загрузите в него содержимое этой папки.
3. Откройте `Settings -> Pages`.
4. В `Build and deployment` выберите:
   - `Source: Deploy from a branch`
   - `Branch: main`
   - `Folder: / (root)`
5. Сохраните настройки и дождитесь публикации.

### Перед публикацией

- Замените `hello@yourdomain.com` в `index.html` на ваш реальный email.
- Замените `https://example.com/` в JSON-LD на будущий адрес сайта.
- Обновите ссылки на ваши соцсети.
